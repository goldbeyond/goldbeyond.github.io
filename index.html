  3. 安装hexo

  4. 初始化hexo

  5. 安装依赖包

  6. 生成静态网页

  7. 运行本地服务

    此时在浏览器中输入：localhost:4000，可查看静态的网页

  8. 添加博文

    刷新上述静态网页，可看到新添加的一篇博文。

    4. 执行npm install hexo-deployer-git --save
